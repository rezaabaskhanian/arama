۲. یه تست ۵ سوالی پر میکنه
۳. بهش میگم «نوع تروما: X»
۴. چندتا تمرین بهش نشون میدم (از فایل تمرین‌ها)
۵. نتیجه توی پروفایلش میمونه
۶. یه صفحه جدا داره برای نوشتن حس و حال روزانه (دفترچه)

فایل‌هایی که گذاشتم:
- سوالات_تست_تروما.json -> سوالای تست
- تمرین‌ها.json -> تمرین برای هر نوع تروما
- قوانین_نمره‌دهی.json -> چطور نوع تروما رو تشخیص بدم
- اسکریپت_پایگاه_داده.sql -> ساختن جدول‌ها

لطفاً اول همین فایل‌ها رو بخون، بعد کدنویسی رو شروع کن.



# تست تشخیص تروما

## تست اصلی
- نام تست: PCL-5 (نسخه استاندارد بین‌المللی)
- تعداد سوالات: ۲۰ سوال
- مقیاس: لیکرت ۵ درجه‌ای (۰ تا ۴)

## نمره‌دهی
- جمع نمرات از ۰ تا ۸۰
- نمره ۳۱-۳۳: آستانه بالینی (احتمال PTSD)

## تفسیر نتایج
- ۰-۲۰: علائم خفیف - تمرین‌های سطح پایه
- ۲۱-۳۲: علائم متوسط - نیاز به تمرین‌های تثبیت و ایمنی
- ۳۳-۵۰: علائم شدید - تمرین‌های تخصصی تروما
- ۵۱-۸۰: علائم خیلی شدید - پیشنهاد مراجعه به متخصص

## منابع تمرین‌ها بر اساس نمره
